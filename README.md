# My_Zootopia

## Project Description
My_Zootopia is a small Python project that fetches animal data from the API Ninjas Animals API and renders the results into an HTML page.

## Features
- Fetch animal data by animal name via external API
- Uses a `.env` file for the API key (no hardcoded secrets)
- Shows a friendly message in the HTML if the animal is not found

## Tech Stack
- Python 3
- requests
- python-dotenv

## Setup

### Requirements
- Python 3
- API Ninjas API key

### Install dependencies
```bash
pip3 install -r requirements.txt
```
